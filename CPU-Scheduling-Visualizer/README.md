# CPU Scheduling and Visualization Tool

## Description
This project demonstrates CPU scheduling algorithms and visualizes them using a Gantt chart.

## Algorithms
- FCFS
- SJF
- Round Robin

## Technologies
- C Programming
- Python
- Matplotlib

## How to Run
1. Compile C program
2. Run it to generate output.txt
3. Run Python script to see Gantt chart

## Output
- Waiting Time
- Turnaround Time
- Gantt Chart visualization
